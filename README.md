
# Aviator Logbook
A lightweight and professional desktop application designed for student pilots and aviation enthusiasts to track flight hours, manage aircraft data, and organize flight logs efficiently.

Key Features
Comprehensive Flight Logging: Record flight dates, routes (ICAO), and durations with ease.

Hour Categorization: Separate tracking for Day/Night, IFR/VFR, and Multi-Engine hours.

Aircraft Management: Maintain a digital inventory of frequently flown aircraft and their technical details.

Data Insights: Automatically calculate total flight time and view performance statistics through a clean dashboard.

Tech Stack
Language: C#

Framework: .NET / WPF (Windows Presentation Foundation)

Database: SQL Server / SQLite

Architecture: MVVM (Model-View-ViewModel)
